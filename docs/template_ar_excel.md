# Plantilla Excel AR (Backend Local Hub Clinico Badajoz)

## Objetivo
Estructura canonica de la hoja `AR` para integrar Primera Visita y Seguimiento en una sola tabla y permitir analisis longitudinal.

## Reglas de codificacion
- `SI`: condicion presente / positivo
- `NO`: condicion ausente / negativo
- `ND`: no determinado / no interrogado / no analizado
- `NA`: no aplica para patologia o tipo de visita
- `""` (vacio): campos de texto libre sin dato

## Estructura
- Bloque base comun: 220 columnas (alineado con ESPA/APS actual).
- Bloque extendido: 101 columnas (multitratamiento + AR especifico).
- Total hoja `AR`: 321 columnas.
